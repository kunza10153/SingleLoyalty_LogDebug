# 📂 JSON to CSV Message Exporter

เครื่องมือสำหรับดึงข้อมูลจากฟิลด์ `message` ภายในไฟล์ JSON (Log format) และส่งออกเป็นไฟล์ CSV เพื่อนำไปใช้งานต่อใน Excel

---

## 🛠 โครงสร้างระบบ (Folder Structure)

จัดเตรียมโฟลเดอร์ให้เป็นระเบียบดังนี้ก่อนใช้งาน:

```
script/
  convert.py   (ไฟล์โปรแกรมหลัก)
  README.md    (ไฟล์นี้)
  input/       (โฟลเดอร์เก็บไฟล์ JSON ต้นทาง)
  output/      (โฟลเดอร์เก็บไฟล์ CSV ที่ได้)
```

---

## 🚀 วิธีการใช้งาน

### 1. เตรียมไฟล์ข้อมูล

นำไฟล์ `.json` ที่คุณต้องการแปลงไปวางไว้ในโฟลเดอร์ `input/`

โปรแกรมรองรับ **หลายไฟล์ JSON** ในการทำงานครั้งเดียว

ตัวอย่าง:

```
input/
  log1.json
  log2.json
  log3.json
```

---

### 2. รันโปรแกรม

#### วิธีที่ 1 (ดับเบิลคลิก)

หากติดตั้ง Python แล้ว สามารถดับเบิลคลิกที่ไฟล์ `convert.py` ได้เลย

#### วิธีที่ 2 (ผ่าน Command Line)

```bash
python3 convert.py
```

---

### 3. รับผลลัพธ์

เมื่อโปรแกรมทำงานเสร็จ จะสร้างไฟล์ CSV แยกตามชื่อ JSON แต่ละไฟล์ในโฟลเดอร์ `output/`

ตัวอย่าง:

```
output/
  log1.csv
  log2.csv
  log3.csv
```

สามารถเปิดไฟล์ด้วย **Microsoft Excel** ได้ทันที (รองรับภาษาไทย ไม่เพี้ยน)

---

## 📝 รายละเอียดการจัดการ Log (Handle Log)

โปรแกรมมีการตรวจสอบข้อผิดพลาดเบื้องต้นดังนี้:

### 🔹 No .json file

หากในโฟลเดอร์ `input/` ไม่มีไฟล์ `.json` โปรแกรมจะแจ้งเตือน

### 🔹 Empty Message

หากไฟล์ JSON ไม่มีฟิลด์ `message` โปรแกรมจะแจ้งเตือนและข้ามไฟล์นั้น

### 🔹 Error per file

หากไฟล์ใดเกิดข้อผิดพลาด โปรแกรมจะไม่หยุดทั้งระบบ แต่จะทำงานต่อกับไฟล์อื่น

---

## ⚙️ รูปแบบ JSON ที่รองรับ

โปรแกรมรองรับโครงสร้าง JSON ที่มีรูปแบบดังนี้:

```json
{
  "hits": {
    "hits": [
      {
        "_source": {
          "message": "log message here"
        }
      }
    ]
  }
}
```
