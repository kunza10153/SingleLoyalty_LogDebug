# 📂 JSON to CSV Message Exporter

เครื่องมือสำหรับดึงข้อมูลจากฟิลด์ `message` ภายในไฟล์ JSON (Log format) และส่งออกเป็นไฟล์ CSV เพื่อนำไปใช้งานต่อใน Excel

## 🛠 โครงสร้างระบบ (Folder Structure)
จัดเตรียมโฟลเดอร์ให้เป็นระเบียบดังนี้ก่อนใช้งาน:
- `script/` (โฟลเดอร์หลัก)
  - `convert.py` (ไฟล์โปรแกรมหลัก)
  - `README.md` (ไฟล์นี้)
  - `input/` (โฟลเดอร์เก็บไฟล์ JSON ต้นทาง)
  - `output/` (โฟลเดอร์เก็บไฟล์ CSV ที่ได้)

## 🚀 วิธีการใช้งาน

1. **เตรียมไฟล์ข้อมูล**: นำไฟล์ `.json` ที่คุณต้องการแปลง (เช่น `log.json`) ไปวางไว้ในโฟลเดอร์ `input/`
   * *หมายเหตุ: โปรแกรมรองรับไฟล์ .json เพียง 1 ไฟล์ในโฟลเดอร์นี้ต่อการทำงาน 1 ครั้ง*

2. **รันโปรแกรม**:
   - **วิธีที่ 1 (ดับเบิลคลิก)**: หากติดตั้ง Python แล้ว สามารถดับเบิลคลิกที่ไฟล์ `convert.py` ได้เลย
   - **วิธีที่ 2 (ผ่าน Command Line)**:
     ```bash
     python3 convert.py
     ```

3. **รับผลลัพธ์**: 
   - เมื่อโปรแกรมทำงานเสร็จ จะปรากฏไฟล์ชื่อ `result.csv` อยู่ในโฟลเดอร์ `output/`
   - สามารถเปิดไฟล์ด้วย **Microsoft Excel** ได้ทันที (ตัวหนังสือภาษาไทยไม่เพี้ยน)

## 📝 รายละเอียดการจัดการ Log (Handle Log)
โปรแกรมมีการตรวจสอบข้อผิดพลาดเบื้องต้นดังนี้:
- **No .json file**: หากในโฟลเดอร์ `input/` ไม่มีไฟล์ .json จะมีการแจ้งเตือน
- **Multiple files**: หากมีไฟล์ .json มากกว่า 1 ไฟล์ โปรแกรมจะหยุดทำงานเพื่อป้องกันการเขียนข้อมูลปนกัน
- **Empty Message**: หากไฟล์ JSON ไม่มีฟิลด์ `message` โปรแกรมจะแจ้งให้ทราบ
